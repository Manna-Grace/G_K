# 🌸 Study Oasis

A brilliant, sweet, and encouraging AI-powered Study Buddy designed to make learning feel like a gentle conversation. 🎀

![Study Oasis UI Demo]

## Project Overview
Study Oasis is an interactive, AI-driven tutoring platform that helps students explore complex subjects through a personalized, friendly interface. Whether you are curious about the mysteries of the universe or the legends of ancient civilizations, Study Oasis is here to guide you with clear explanations, fun quizzes, and a touch of elegance.

## Key Features
* **Intelligent Tutoring:** Powered by a customized AI agent designed for a friendly, supportive personality.
* **Knowledge Retrieval:** Seamlessly integrates with educational vector stores to provide accurate, factual notes.
* **Interactive Quiz Mode:** Engage in fun, 3-question multiple-choice quizzes to test your knowledge.
* **Elegant UI:** Built with Material Design principles, featuring glassmorphism effects, smooth gradients, and a soothing aesthetic.
* **Context-Aware Learning:** Tailors explanations based on the subject you choose to focus on.

## Tech Stack
* **Language:** Python
* **AI Framework:** PydanticAI & Groq (Llama 3.1)
* **Frontend/UI:** Chainlit
* **Styling:** Custom CSS
* **Data Processing:** Vector stores with RAG
